# Video View

This view allows you to preview and copy the video files.

![github pages](_images/15.png)

- 1. Chose Video View.
- 2. Preview Panel
- 3. Show video in explorer.
- 4. Select Previous Video.
- 5. Play Video.
- 6. Select Next Video.
- 7. Copy video to `clipboard`.
- 8. List of Videos in current category.
